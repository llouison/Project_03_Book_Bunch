Book Bunch API

Hello World! Book Bunch is a full-stack single-paged virtual book shelf CRUD app. It was created by Ahsan Abassi, Lisa Louison, and Aliaksei Babkou as a part of the Web Development Immersive course at General Assembly NYC. This is a REST-ful API that serves the list of books added to the Book Bunch database.

No keys are needed to access this API.

Endpoints

The API has two main endpoints: /api/books and /api/users. Here are some sample calls from each and their responses.

Other Endpoints

POST to api/quotes

Adds a quote to the database. Request body must include content, author, and genre_id properties.

fetch('https://ada-api.herokuapp.com/api/quotes', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    content: e.target.content.value,
    author: e.target.author.value,
    genre_id: e.target.genre_id.value,
  }),
})

PUT to api/quotes/:id

Modify a quote’s information in the database.

// where ID is the quote's ID
fetch(`https://ada-api.herokuapp.com/api/quotes/${id}`, {
  method: 'PUT',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    content: 'quote content here',
    author: 'quote author here',
    genre_id: 3 // the genre ID must be a number
  }),
})

DELETE to api/quotes/:id

Remove a quote from the database.

// where ID is the quote's ID
fetch(`https://ada-api.herokuapp.com/api/quotes/${id}`, {
  method: 'DELETE',
})
